# Sample packet datasets

This directory is reserved for small, anonymised demonstration datasets that are safe to publish.

Do not add:

- Real customer or employer network data
- Raw PCAP or PCAPNG captures
- Credentials, tokens, cookies, or payload content
- Public or private IP addresses that identify a sensitive environment
- Hostnames, MAC addresses, usernames, or device names from real systems

Any sample CSV added here should include a short note describing:

- How the data was generated
- Which Wireshark export profile it follows
- Which fields are included
- Which anonymisation steps were applied
- What analysis behaviour the sample is intended to demonstrate
